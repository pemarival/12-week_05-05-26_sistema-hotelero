# Sistema de hotelería — Posibles funcionalidades del sistema

## 1. Módulo: Parametrización

### Funcionalidad: Gestión de cliente
Permite registrar, consultar, actualizar y controlar la información de cliente que realiza reserva, estadía, compra de producto o compra de servicio.

### Funcionalidad: Gestión de precio dinámico
Permite configurar precios variables según tipo de habitación, tipo de día, temporada, feriado, día entre semana o condición especial definida por el negocio.

### Funcionalidad: Registro de empresa
Permite administrar los datos generales del hotel o empresa hotelera, incluyendo información comercial, contacto y datos requeridos para facturación.

### Funcionalidad: Gestión de información legal
Permite registrar datos legales asociados a la operación del hotel, documentos comerciales, políticas internas y condiciones aplicables al servicio.

### Funcionalidad: Gestión de empleado
Permite administrar la información del empleado que participa en procesos de recepción, operación, mantenimiento, inventario, facturación o administración.

### Funcionalidad: Gestión de tipo de día
Permite clasificar el calendario operativo en temporada, feriado, día entre semana, fin de semana o día especial, para soportar reglas de precio.

### Funcionalidad: Gestión de método de pago
Permite parametrizar los métodos de pago aceptados por el hotel para reserva, abono, pago parcial o factura final.

## 2. Módulo: Distribución

### Funcionalidad: Gestión de sede
Permite registrar y administrar las sedes del hotel, incluyendo ubicación, capacidad operativa y relación con habitaciones.

### Funcionalidad: Gestión de habitación
Permite administrar cada habitación del hotel, su identificación, sede, tipo, capacidad, estado y disponibilidad operativa.

### Funcionalidad: Gestión de tipo de habitación
Permite definir categorías de habitación, como sencilla, doble, suite u otras clasificaciones comerciales del negocio.

### Funcionalidad: Gestión de estado de habitación
Permite controlar si una habitación se encuentra disponible, reservada, ocupada, en limpieza, bloqueada o en mantenimiento.

## 3. Módulo: Prestación de servicio

### Funcionalidad: Consulta de catálogo de habitación
Permite visualizar las habitaciones disponibles para oferta comercial, junto con sus características principales, tipo y precio estimado.

### Funcionalidad: Consulta de disponibilidad de habitación
Permite validar si una habitación se encuentra disponible en un rango de fechas específico.

### Funcionalidad: Gestión de reserva de habitación
Permite crear y administrar una reserva asociada a cliente, habitación, fechas, precio y estado de reserva.

### Funcionalidad: Cancelación de reserva de habitación
Permite cancelar una reserva de acuerdo con las condiciones definidas por el hotel.

### Funcionalidad: Check in
Permite registrar el ingreso del cliente al hotel, activar la estadía y actualizar el estado de la habitación.

### Funcionalidad: Check out
Permite registrar la salida del cliente, cerrar la estadía, liquidar consumos y actualizar el estado de la habitación.

### Funcionalidad: Secuencia de estadía
Permite registrar los eventos ocurridos durante la estadía, incluyendo consumo de producto, consumo de servicio y novedades operativas.

### Funcionalidad: Venta de producto
Permite registrar productos consumidos por el cliente durante la estadía.

### Funcionalidad: Venta de servicio
Permite registrar servicios adicionales solicitados por el cliente durante la estadía.

## 4. Módulo: Facturación

### Funcionalidad: Pre facturación
Permite generar una vista preliminar de valores por reserva, estadía, producto y servicio antes de emitir la factura final.

### Funcionalidad: Seguimiento de pago parcial
Permite controlar abono, pago parcial, saldo pendiente y estado de pago asociado a reserva o factura.

### Funcionalidad: Gestión de factura
Permite emitir y consultar la factura definitiva del cliente.

### Funcionalidad: Detalle de compra
Permite relacionar producto y servicio consumido con la factura o pre factura correspondiente.

## 5. Módulo: Inventario

### Funcionalidad: Gestión de producto
Permite registrar producto disponible para venta o consumo interno del hotel.

### Funcionalidad: Gestión de servicio
Permite registrar servicio adicional disponible para los clientes.

### Funcionalidad: Gestión de proveedor
Permite administrar proveedor relacionado con abastecimiento de producto o prestación de servicio.

### Funcionalidad: Seguimiento de producto
Permite controlar entradas, salidas, consumo y trazabilidad de producto.

### Funcionalidad: Consulta de disponibilidad
Permite conocer la disponibilidad actual de producto o servicio antes de venderlo.

## 6. Módulo: Notificación

### Funcionalidad: Promoción
Permite enviar información comercial al cliente sobre ofertas, descuentos o campañas.

### Funcionalidad: Alerta
Permite enviar mensajes relacionados con reserva, pago, check in, check out, disponibilidad o novedades.

### Funcionalidad: Término y condición
Permite presentar condiciones antes de la compra de producto o servicio.

### Funcionalidad: Fidelización de cliente
Permite gestionar comunicaciones orientadas a retención, beneficios y recompra.

## 7. Módulo: Seguridad

### Funcionalidad: Registro de persona
Permite almacenar información base de personas asociadas al sistema.

### Funcionalidad: Registro de usuario
Permite crear credenciales de acceso para operar el sistema.

### Funcionalidad: Gestión de rol
Permite definir perfiles de acceso según responsabilidad.

### Funcionalidad: Gestión de permiso
Permite controlar acciones autorizadas dentro del sistema.

### Funcionalidad: Gestión de módulo
Permite organizar las funcionalidades del sistema por dominio funcional.

### Funcionalidad: Gestión de vista
Permite controlar el acceso visual o funcional a pantallas específicas.

## 8. Módulo: Mantenimiento

### Funcionalidad: Mantenimiento de habitación por uso
Permite registrar actividades de mantenimiento derivadas del uso normal de la habitación.

### Funcionalidad: Mantenimiento por remodelación
Permite bloquear y controlar habitaciones durante procesos de remodelación.

### Funcionalidad: Dashboard operativo
Permite visualizar disponibilidad, mantenimiento y seguimiento de habitaciones.
